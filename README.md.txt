This project demonstrates real-time hand gesture recognition using
MediaPipe and OpenCV. The system detects hand landmarks from webcam
input and classifies simple gestures mapped to demo actions.



Features
- Real-time hand detection
- MediaPipe hand landmark tracking
- Gesture classification
- Gesture-to-action mapping
- Webcam-based demo



Gestures Supported
| Gesture     | Action        |
|------------|---------------|
| Thumbs Up  | Volume Up     |
| Fist       | Pause         |
| Open Palm  | Play          |



Technologies Used
- Python
- OpenCV
- MediaPipe



How to Run
```bash
pip install -r requirements.txt
python src/gesture_recognition.py
